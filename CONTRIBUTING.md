The goal of this project is to create a comprehensive, high-quality resource for computer science students and enthusiasts.

## How to Contribute

### 1. Reporting Bugs
If you find a mistake (typo, broken link, incorrect information), please open an issue.
- Use a clear and descriptive title.
- Describe the error and where it is located.
- Suggest a fix if possible.

### 2. Suggesting Enhancements
Have an idea for a new roadmap or section?
- Open an issue to discuss it first.
- Explain why it would be useful.

### 3. Pull Requests
1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  Create a new **branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-new-content
    ```
4.  **Commit** your changes with meaningful messages.
5.  **Push** to your branch.
6.  Open a **Pull Request** (PR).

## Style Guide

- **Markdown**: Use standard Markdown.
- **Language**: We support both English and Arabic. Please try to keep translations synchronized if possible.
- **Structure**: Follow the existing directory structure.
- **Tone**: Keep it encouraging, educational, and professional.

## Code of Conduct

Please be respectful and kind to other contributors. We are here to learn and grow together.
