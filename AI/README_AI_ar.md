# دليل الذكاء الاصطناعي — AI (النسخة العربية)

> هذا الملف هو المدخل الرئيسي المبسّط لمجلد `AI/`. يقدّم نظرة عامة مركّزة على التخصصات الأساسية في الذكاء الاصطناعي وروابط مباشرة للملفات المفصّلة (النسخة العربية لكل تخصص). هذا README مُصمّم للمبتدئين وطلاب علوم الحاسب والراغبين بالانتقال إلى تخصص جديد داخل مجال AI.

---

## ما الذي يحتويه هذا الدليل؟
يتضمن هذا الدليل **7 تخصصات رئيسية** ومركزيّة في مجال الذكاء الاصطناعي، كل تخصص لديه ملف Markdown مفصّل يغطي: تعريف التخصص، مهام المختص، الفروع الداخلية، مفاهيم أساسية، أدوات وتقنيات، سيناريو عملي معمّق، الأدوار الوظيفية، ومسرد المصطلحات.

**التخصصات السبع الأساسية:**
1. التعلّم الآلي (Machine Learning)  
2. التعلّم العميق (Deep Learning)  
3. معالجة اللغة الطبيعية (NLP)  
4. الرؤية الحاسوبية (Computer Vision)  
5. هندسة تعلم الآلة وتشغيل النماذج (MLOps & ML Engineering)  
6. التعلّم المعزّز (Reinforcement Learning)  
7. أمان الذكاء الاصطناعي وشرح النماذج (AI Safety & Explainability)

## كيف تستخدم هذا الملف؟
1. ابدأ بقراءة **الملف العام** ثم اختر تخصصًا واحدًا لقراءته بتفصيل. الهدف أن تفهم الصورة الكاملة أولًا ثم تتعمق في تخصص واحد.  
2. كل ملف تخصص يحتوي على **سيناريو عملي** يمكنك تنفيذه كمشروع محفظة (Portfolio). ابدأ بواحد ثم انتقل للآخر.  
3. اتّبع مسارات التعلم المذكورة داخل كل ملف (من مبتدئ إلى متقدم).  
4. استخدم نسخ الـ `.md` الإنجليزية إذا رغبت في الاطّلاع على المصطلحات التقنية باللغة الإنجليزية (يمكن إضافتها لاحقًا إلى نفس المجلد).

## نصائح لترتيب الريبو على GitHub
1. أنشئ مجلدًا اسمه `AI/` في الريبو.  
2. ضع كل ملفات التخصصات السبعة داخل هذا المجلد بالأسماء المشار إليها أعلاه.  
3. أضف الملف `README_ar.md` (الذي تقرأه الآن) إلى نفس المجلد.  
4. افعل `git add . && git commit -m "Add AI core specialties (AR) and README" && git push` لرفع التغييرات.  
5. يفضّل إنشاء فرع (branch) جديد لإجراء التعديلات ثم فتح Pull Request لمراجعتها إن كنت تعمل مع فريق.

---

## اقتراح مسار تعلّم للمبتدئين (موجز)
1. تعلّم أساسيات Python وGit وSQL.  
2. ابدأ بـ **التعلّم الآلي التقليدي (ML)**: scikit-learn، مفاهيم الإحصاء الأساسية، مشاريع بسيطة.  
3. بعدها انتقل إلى **التعلّم العميق (DL)**: PyTorch أو TensorFlow، مشاريع على الصور/النصوص.  
4. جرّب مشاريع صغيرة في **NLP** و**CV** (بوت دردشة بسيط، نموذج كشف كائنات).  
5. تعرّف على **MLOps** مبكرًا: كيف تُسلم نموذجًا وتراقبه في الإنتاج.  
6. بعد بناء أساس قوي، استكشف **RL** و**AI Safety** كمسارات متقدمة.
